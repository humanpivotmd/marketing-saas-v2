# 업종별 콘텐츠 전략 가이드

> `pipeline.ts`의 `getIndustryContext()` 함수에 구현된 14개 대분류, 37개 중분류 업종별 콘텐츠 전략. 마이페이지에서 업종을 설정하면 제목 생성(STEP3), 초안(STEP4), 채널별 변환(STEP5) 모든 단계에 자동 반영된다.

---

## 1. 동작 원리

### 프롬프트 주입 구조

마이페이지에서 설정한 `industry` 값이 `PipelineContext.industry`로 전달되면, `getIndustryContext()` 함수가 해당 업종의 맞춤 컨텍스트를 프롬프트에 주입한다.

```
[업종 맞춤 컨텍스트 - {업종명}]
- 핵심 가치: {coreValue}
- 주요 고객: {audience}
- 필수 요소: {essentials}
- 차별화 포인트: {differentiator}
- 감정 트리거: {emotionTrigger}     // 있는 경우
- 주의사항: {caution}               // 있는 경우
```

### 적용 범위

| 단계 | 적용 내용 |
|------|-----------|
| STEP3 제목 생성 | 업종 핵심 가치 + 차별화 포인트 반영 |
| STEP4 초안 작성 | 필수 요소 + 고객 타겟 반영 |
| STEP5 채널별 변환 | 업종 컨텍스트 전체 반영 |
| STEP6 이미지 | 업종에 맞는 비주얼 가이드 (간접 반영) |

## 4. 업종 미설정 시 동작

`industry`가 설정되지 않았거나 매칭되는 업종 데이터가 없는 경우:

- `getIndustryContext()`가 빈 문자열 또는 `"- 업종: {입력값}"` 만 반환
- 범용적인 B2B/B2C 프롬프트만 적용 (업종 특화 없음)
- 마이페이지에서 업종을 정확히 설정하면 콘텐츠 품질이 크게 향상됨
