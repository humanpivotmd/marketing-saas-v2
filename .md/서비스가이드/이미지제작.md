# STEP6: 이미지 프롬프트 가이드

> 콘텐츠 본문을 기반으로 AI 이미지 생성 도구(Midjourney, DALL-E 등)에서 바로 사용할 수 있는 프롬프트를 생성한다. 본문 이미지 3장 + 썸네일 1장.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| 파이프라인 위치 | STEP6 (STEP5 채널별 변환 완료 후) |
| API | `/api/generate/image-script` |
| AI 모델 | `claude-sonnet-4-20250514` / max_tokens: 2048 |
| 출력 | 이미지 프롬프트 3개 + 썸네일 프롬프트 1개 (한글 설명 + 영문 프롬프트) |
| DB 저장 | `image_scripts` 테이블 |

---

## 2. API 상세

### `/api/generate/image-script`

```
POST /api/generate/image-script
Content-Type: application/json
Authorization: Bearer {token}

{
  "project_id": "uuid-...",
  "channel": "blog",
  "ai_tool": "Midjourney",
  "image_size": "1080x1080",
  "image_style": "photo",
  "style_detail": "미니멀"
}
```

**요청 파라미터:**

| 필드 | 필수 | 기본값 | 설명 |
|------|------|--------|------|
| project_id | O | - | 프로젝트 ID |
| channel | O | - | 대상 채널 (blog, threads, instagram, facebook) |
| ai_tool | - | Midjourney | AI 이미지 도구 |
| image_size | - | 1080x1080 | 이미지 사이즈 |
| image_style | - | photo | `photo` (실사) 또는 `illustration` (일러스트) |
| style_detail | - | 미니멀 | 세부 스타일 키워드 |

**중복 방지:** 같은 프로젝트 + 같은 채널의 기존 이미지 스크립트를 삭제 후 재생성한다.

## 4. 이미지 스타일

### photo (실사 사진)

자연스러운 사진 느낌. 제품/서비스/공간 촬영에 적합.

### illustration (일러스트/그림)

그래픽 디자인 느낌. 개념 설명, 인포그래픽, 추상적 콘텐츠에 적합.

### 세부 스타일 (style_detail) 예시

| 스타일 | 설명 | 적합 업종 |
|--------|------|-----------|
| 미니멀 | 깔끔하고 여백이 많은 디자인 | IT/SaaS, 전문직 |
| 모던 | 현대적이고 트렌디한 느낌 | 뷰티, 패션, 카페 |
| 따뜻한 | 따뜻한 색감, 감성적 분위기 | 음식점, 교육, 종교 |
| 전문적 | 비즈니스 톤, 차분한 색상 | B2B, 컨설팅, 의료 |
| 빈티지 | 복고풍, 클래식 느낌 | 전통음식, 인테리어 |
| 팝아트 | 밝고 강렬한 색상 | 엔터테인먼트, 소매 |

---

## 5. B2B vs B2C 이미지 전략

### B2B 이미지 원칙

| 요소 | 가이드 |
|------|--------|
| 분위기 | 전문적이고 깔끔한 비즈니스 톤 |
| 주요 소재 | 데이터 차트, 인포그래픽, 대시보드 화면 |
| 장면 | 회의실, 오피스, 비즈니스 미팅 |
| 색상 | 차분한 블루/그레이 계열 |
| 텍스트 오버레이 | 수치, 퍼센트, 성과 지표 |

### B2C 이미지 원칙

| 요소 | 가이드 |
|------|--------|
| 분위기 | 감성적이고 밝은 라이프스타일 |
| 주요 소재 | 인물 중심, 일상 속 사용 장면 |
| 장면 | 카페, 집, 야외, 매장 등 일상 공간 |
| 색상 | 따뜻한 자연광, 생동감 있는 색상 |
| 텍스트 오버레이 | 감성 카피, 혜택 강조 |

## 6. 채널별 이미지 규격

| 채널 | 권장 사이즈 | 비율 | 특징 |
|------|------------|------|------|
| 블로그 | 1200x675 | 16:9 | 본문 삽입 + 썸네일 |
| Threads | 1080x1080 | 1:1 | 정사각형, 텍스트 최소화 |
| 인스타그램 | 1080x1080 | 1:1 또는 4:5 | 피드 최적화, 비주얼 중심 |
| 페이스북 | 1200x630 | 약 1.9:1 | 링크 공유 시 미리보기 |

## 7. 업종별 이미지 전략

### 음식업

| 업종 | 이미지 핵심 | 촬영 포인트 |
|------|------------|-------------|
| 한식/전통음식 | 상차림 전체, 반찬 클로즈업 | 정감 있는 한옥/전통 느낌 |
| 카페/디저트 | 인테리어 + 시그니처 메뉴 | 밝은 자연광, 인스타감성 |
| 일반 음식점 | 대표 메뉴 + 매장 분위기 | 식욕 자극 색감 |

### 의료

| 업종 | 이미지 핵심 | 주의사항 |
|------|------------|----------|
| 병원/의원 | 의료진, 깨끗한 시설 | 의료법 준수, 과대광고 금지 |
| 치과 | 최신 장비, 밝은 진료실 | 시술 전후 사진 주의 |
| 한의원 | 한약재, 전통+현대 조화 | 자연 치유 이미지 |

### 뷰티/헬스

| 업종 | 이미지 핵심 | 촬영 포인트 |
|------|------------|-------------|
| 미용실 | 시술 전후, 스타일리스트 | 트렌디한 컬러/조명 |
| 네일/피부관리 | 시술 클로즈업 | 섬세함과 청결함 |
| 헬스/피트니스 | 시설, 트레이닝 장면 | 활력과 변화의 이미지 |

### IT/전문직

| 업종 | 이미지 핵심 | 촬영 포인트 |
|------|------------|-------------|
| SW/앱 개발 | 대시보드, 코드 화면 | 기술적이면서 세련된 |
| IT SaaS | 제품 UI, 데이터 차트 | 모던, 미니멀 |
| 컨설팅 | 미팅 장면, 보고서 | 전문적 신뢰감 |

## 8. 이미지 배치 가이드 (placement)

| 위치 | 역할 | 추천 이미지 |
|------|------|-------------|
| 본문 상단 | 첫인상, 주제 소개 | 전체적인 분위기/장면 |
| 본문 중간 | 핵심 포인트 시각화 | 제품/서비스 클로즈업, 데이터 |
| 본문 하단 | 마무리, CTA 보조 | 행동 유도 이미지 |
| 썸네일 | 검색 결과 노출 | 텍스트 공간 확보, 핵심 비주얼 |

## 9. DB 스키마 (image_scripts)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| project_id | uuid | 프로젝트 FK |
| user_id | uuid | 사용자 FK |
| content_id | uuid | 콘텐츠 FK (nullable) |
| channel | text | 대상 채널 |
| ai_tool | text | AI 도구명 |
| image_size | text | 이미지 사이즈 |
| image_style | text | photo / illustration |
| style_detail | text | 세부 스타일 |
| prompts | jsonb | 이미지 프롬프트 배열 |
| thumbnail_prompt | text | 썸네일 프롬프트 (JSON) |
| status | text | generated / used |
