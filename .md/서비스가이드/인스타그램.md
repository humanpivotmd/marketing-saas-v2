# 인스타그램 채널 가이드

> STEP5 파이프라인에서 블로그 초안을 인스타그램 캡션으로 변환하고, Instagram Graph API를 통해 직접 발행한다.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| 채널 변환 | `/api/generate/pipeline` (channel: instagram) |
| 발행 API | `/api/publish/instagram` |
| AI 모델 | `claude-sonnet-4-20250514` / max_tokens: 2048 |
| Instagram API | Graph API v19.0 (graph.facebook.com) |
| 이미지 필수 | 발행 시 이미지 URL 필수 (텍스트만 발행 불가) |

---

## 3. 변환 프로세스

```
블로그 초안 (draftContent) 또는 블로그 완성본 (blogContent)
  ↓
buildChannelPrompt(ctx, 'instagram')
  ↓
Claude API → 인스타그램 캡션 텍스트
  ↓
contents 테이블 저장 (type: 'instagram')
  ↓
해시태그 자동 추출 → contents.hashtags에 저장
```

## 4. Instagram Graph API 발행

### API: `/api/publish/instagram`

```
POST /api/publish/instagram
Content-Type: application/json
Authorization: Bearer {token}

{
  "content_id": "uuid-...",
  "image_url": "https://publicly-accessible-image-url.jpg"  // 필수
}
```

**요청 파라미터:**

| 필드 | 필수 | 타입 | 설명 |
|------|------|------|------|
| content_id | O | uuid | 발행할 콘텐츠 ID |
| image_url | O | URL | 공개 접근 가능한 이미지 URL |

**주의:** Instagram은 이미지 없이 텍스트만 발행할 수 없다. `image_url`이 없으면 `"Instagram 발행에는 이미지가 필요합니다."` 에러 반환.

### 발행 플로우 (2단계)

#### Step 1: 미디어 컨테이너 생성

```
POST https://graph.facebook.com/v19.0/{ig_user_id}/media
{
  "caption": "캡션 텍스트 + 해시태그...",
  "image_url": "https://...",
  "access_token": "..."
}
```

#### Step 2: 미디어 발행

```
POST https://graph.facebook.com/v19.0/{ig_user_id}/media_publish
{
  "creation_id": "container_id",
  "access_token": "..."
}
```

### 사전 조건

| 조건 | 설명 |
|------|------|
| 채널 연동 | `channels` 테이블에 platform='instagram' 레코드 필요 |
| access_token | Instagram Graph API 장기 토큰 (Facebook 비즈니스 계정 경유) |
| platform_user_id | Instagram 비즈니스 계정 ID |
| 이미지 | 공개 접근 가능한 URL (HTTPS) |

연동되지 않은 경우 `"Instagram 계정이 연동되지 않았습니다. 설정에서 연동해주세요."` 에러 반환.

### 발행 후 처리

발행 성공 시:
- `contents.status` → `'published'`
- `contents.published_at` → 현재 시각
- `contents.meta.platform_post_id` → Instagram 게시물 ID
- `contents.meta.platform` → `'instagram'`

## 5. B2B 인스타그램 전략

### 콘텐츠 유형별 가이드

| 유형 | 구성 | 이미지 |
|------|------|--------|
| 인사이트 카드 | 핵심 데이터 + 해석 | 인포그래픽, 차트 |
| 사례 소개 | 고객 성공 스토리 | 결과 스크린샷, 로고 |
| 팁 리스트 | 3~5개 실행 가능한 팁 | 넘버링 카드 이미지 |
| 업계 뉴스 | 트렌드 분석 + 시사점 | 뉴스 카드 형태 |

### B2B 해시태그 전략

```
[업계 전문 태그] 5~7개
#마케팅자동화 #B2B마케팅 #SaaS #마테크

[일반 비즈니스 태그] 3~5개
#비즈니스성장 #디지털전환 #스타트업

[행동 유도 태그] 2~3개
#마케팅팁 #인사이트 #마케팅공부
```

## 6. B2C 인스타그램 전략

### 콘텐츠 유형별 가이드

| 유형 | 구성 | 이미지 |
|------|------|--------|
| 감성 스토리 | 경험/후기 형식 | 라이프스타일 사진 |
| 꿀팁 리스트 | 숫자 + 간결한 팁 | 카드 뉴스 형태 |
| 비포/애프터 | 변화 전후 비교 | 비교 사진 |
| 이벤트/혜택 | 할인/특가 안내 | 프로모션 이미지 |

### B2C 해시태그 구조 (15~25개)

```
[핵심 태그] 3~5개 (직접 관련)
#카페인테리어 #카페창업 #카페디자인

[확장 태그] 5~8개 (관련 주제)
#인테리어팁 #소자본창업 #공간디자인 #인테리어참고

[감성/트렌드 태그] 3~5개
#감성카페 #카페스타그램 #인스타카페

[발견 태그] 3~5개 (검색용)
#카페추천 #카페투어 #카페맛집
```

## 8. 이미지 규격

| 포맷 | 사이즈 | 비율 | 용도 |
|------|--------|------|------|
| 정사각형 | 1080x1080 | 1:1 | 피드 기본 |
| 세로형 | 1080x1350 | 4:5 | 피드 노출 면적 최대 |
| 가로형 | 1080x566 | 1.91:1 | 풍경/전경 |
| 스토리/릴스 | 1080x1920 | 9:16 | 풀스크린 세로 |

**권장:** 피드 게시물은 4:5 세로형이 노출 면적이 가장 크다.
