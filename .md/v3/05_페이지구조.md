# v3 페이지 구조

## 1. 라우트 맵

```
(dashboard)
├── /create/                          (layout.tsx: Stepper + children)
│   ├── /create/draft-info            STEP3 초안 정보 입력
│   ├── /create/generating            STEP4 초안 생성 (로딩)
│   ├── /create/channel-write         STEP5 채널별 콘텐츠
│   ├── /create/image-script          STEP6 이미지 프롬프트
│   └── /create/video-script          STEP7 영상 스크립트
├── /settings                         설정 (BusinessProfileTab = STEP1)
├── /contents                         콘텐츠 목록 (프로젝트 탭 포함)
├── /calendar                         캘린더
└── /keywords                         STEP2 키워드 분석 (기존)

(public)
├── /landing                          랜딩 페이지
└── /guide                            사용 가이드

(admin)
└── /admin/industries                 업종 관리 (관리자)
```

## 2. 각 페이지 상세

### /create/ (layout.tsx) - Stepper 구조

Stepper 단계 표시:
| 표시 번호 | 라벨 | 경로 |
|-----------|------|------|
| 2 | 키워드 | /keywords |
| 3 | 초안 정보 | /create/draft-info |
| 4 | 초안 작성 | /create/generating |
| 5 | 채널별 글 | /create/channel-write |
| 6 | 이미지 | /create/image-script |
| 7 | 영상 | /create/video-script |

- 현재 경로 기준으로 isCurrent/isCompleted/isPending 상태 표시
- 완료 단계: 녹색 체크, 현재 단계: 액센트 색상, 미완료: 회색
- 가로 스크롤 지원 (scrollbar-hide)

### /create/draft-info (STEP3)

**기능:**
- 마이페이지 설정 자동 로드 (business_type, company_name, service_name, writing_tone)
- B2B/B2C 선택 (2개 버튼)
- 회사명/서비스명 입력
- 글 주제 유형 4종 선택 (info/intro/service/product)
- 톤앤매너 5종 선택 (auto/professional/friendly/formal/conversational)
- 커스텀 프롬프트 textarea + 3모드 선택 (combine/priority/reference)
- "제목 생성하기" 버튼 -> 제목 5개 표시 -> 1개 선택
- "초안 작성 시작" 버튼 -> 프로젝트 POST -> PATCH(STEP3 데이터) -> /create/generating 이동

**API 호출:**
- GET /api/mypage/business-profile (마운트 시)
- POST /api/generate/titles (제목 생성)
- POST /api/projects (프로젝트 생성)
- PATCH /api/projects/:id (STEP3 데이터 저장)

**가드:** SetupRequired (마이페이지 설정 필수)

### /create/generating (STEP4)

**기능:**
- 로딩 애니메이션 + 프로그레스 바 (0~85% 애니메이션 -> 100%)
- 초안 생성 API 자동 호출
- 완료 시 1초 후 /create/channel-write로 자동 이동
- 에러 시 "이전으로" / "다시 시도" 버튼

**API 호출:**
- POST /api/generate/draft

**가드:** FlowGuard (projectId, requiredStep=4)

### /create/channel-write (STEP5)

**기능:**
- 기존 콘텐츠가 없으면 자동으로 SSE 생성 시작
- 채널별 프로그레스 표시 (SSE 이벤트 수신)
- 채널 탭 UI (블로그/Threads/인스타그램/페이스북)
- 각 채널별: 본문 표시, 글자수, 해시태그, 복사 버튼
- 수정 요청 textarea + "수정 반영하여 재작성" (POST /api/generate/single)
- 발행 예정일 date picker
- "전체 컨펌 -> 이미지" 버튼: 모든 콘텐츠 status=confirmed, 프로젝트 confirmed_at, current_step=6

**API 호출:**
- GET /api/projects/:id (프로젝트 + 콘텐츠 조회)
- POST /api/generate/pipeline (SSE 스트림)
- POST /api/generate/single (재작성)
- PUT /api/contents/:id (상태 업데이트)
- PATCH /api/projects/:id (컨펌)

**가드:** FlowGuard (projectId, requiredStep=5)

### /create/image-script (STEP6)

**기능:**
- AI 도구 선택: Midjourney, DALL-E, Stable Diffusion, Gemini, Ideogram, Flux
- 이미지 유형: photo(실사) / illustration(일러스트)
- 스타일: 미니멀/모던/빈티지/팝아트/수채화/3D/플랫/도시적/세련된
- 채널별 이미지 사이즈 자동 매핑:
  - blog: 1200x630, 800x800
  - instagram: 1080x1080, 1080x1920
  - threads: 1080x1080
  - facebook: 1200x630, 1080x1920
- "프롬프트 생성" -> 모든 채널에 대해 순차 호출
- 결과: 채널별 카드에 이미지 프롬프트 3개 + 썸네일 표시
- 각 프롬프트: 한글 설명 + 영문 프롬프트(code 블록) + 복사 버튼
- "다음 -> 영상" 버튼: current_step=7, s6=completed, /create/video-script 이동

**API 호출:**
- GET /api/projects/:id
- POST /api/generate/image-script (채널별)
- PATCH /api/projects/:id

**가드:** FlowGuard (projectId, requiredStep=6)

### /create/video-script (STEP7)

**기능:**
- 영상 유형: short(숏폼 세로, 60초 이내) / normal(일반 가로, 3~5분)
- 대상 채널: youtube, instagram_reels, tiktok
- 장면 수: 1~5 (range slider)
- 장면당 시간: 3~10초 (range slider)
- 이미지 유형: photo/illustration
- 스타일: 미니멀/모던/빈티지/도시적/세련된/따뜻한/전문적
- "영상 스크립트 생성" -> 결과 스토리보드 카드 표시
  - 영상 제목, 총 시간, 장면 수
  - 후킹 멘트 (있으면)
  - 장면별: 화면 설명, 나레이션, 자막, 전환 효과, 시간
  - BGM 제안 (있으면)
- "프로젝트 완료" 버튼: status=completed, s1~s7=completed -> /contents 이동

**API 호출:**
- GET /api/projects/:id
- POST /api/generate/video-script
- PATCH /api/projects/:id

**가드:** FlowGuard (projectId, requiredStep=7)

### /settings (BusinessProfileTab)

**기능** (마이페이지 탭 = STEP1):
- B2B/B2C 비즈니스 유형 선택
- 업종 선택 (industries API, 트리 구조 optgroup)
- 회사명 / 서비스명 입력
- 운영 채널 복수 선택 (blog/threads/instagram/facebook/video)
- 타겟 고객 텍스트 입력
- 성별 선택 (all/male/female)
- 고정 키워드 최대 5개 (추가/삭제)
- 네이버 블로그 주제 33개 선택
- 글 톤 선택
- "저장" 버튼 -> PUT /api/mypage/business-profile

설정 탭 전체:
| 탭 ID | 라벨 | 설명 |
|--------|------|------|
| business | 마이페이지 | v3 비즈니스 프로필 (위 내용) |
| profile | 프로필 | 이름/비밀번호/계정 삭제 |
| plan | 플랜 | 현재 플랜/업그레이드 |
| keyword-settings | 키워드 설정 | 키워드 분석 관련 설정 |
| prompts | 채널 프롬프트 | 채널별(blog/instagram/threads/facebook/image/video_script) 커스텀 프롬프트 |
| brand-voice | Brand Voice | 브랜드 보이스 관리 |
| sns | SNS 연동 | SNS 계정 연동 |
| notifications | 알림 | 알림 설정 |

### /contents

**기능:**
- 채널 탭 필터: 전체, 프로젝트, 블로그, Threads, 인스타그램, 페이스북, 영상 스크립트
- "프로젝트" 탭: GET /api/projects로 프로젝트 목록 표시
- 나머지 탭: GET /api/contents로 콘텐츠 목록 표시
- 검색, 정렬(생성일/제목/SEO점수), 페이지네이션

### /calendar

**기능:**
- 월간 캘린더 그리드 (7x6)
- 채널별 색상 점 표시 (blog=녹색, threads=보라, instagram=핑크, facebook=파랑, video_script=주황)
- 날짜 클릭 시 사이드바에 해당 날짜 콘텐츠 목록
- 이번 달 채널별 콘텐츠 수 요약
- 월 이동 (이전/다음), "오늘" 버튼
- API: GET /api/calendar?from=YYYY-MM-DD&to=YYYY-MM-DD

## 3. FlowGuard / SetupRequired 적용 위치

| 컴포넌트 | 적용 페이지 | 조건 |
|----------|------------|------|
| SetupRequired | /create/draft-info | business_type + selected_channels + company_name 설정 여부 확인 |
| FlowGuard | /create/generating | projectId 필수, requiredStep=4 |
| FlowGuard | /create/channel-write | projectId 필수, requiredStep=5 |
| FlowGuard | /create/image-script | projectId 필수, requiredStep=6 |
| FlowGuard | /create/video-script | projectId 필수, requiredStep=7 |

### SetupRequired 동작
1. sessionStorage에 `business_setup=done`이면 통과
2. GET /api/mypage/business-profile 호출
3. business_type + selected_channels.length > 0 + company_name 모두 있으면 통과
4. 미설정 시: "마이페이지 설정이 필요합니다" 안내 + /settings#business 링크

### FlowGuard 동작
1. projectId 없으면: "프로젝트 정보가 없습니다" + /keywords 링크
2. GET /api/projects/:id로 current_step 확인
3. current_step < requiredStep이면: "이전 단계를 먼저 완료해주세요" + 이전 버튼
4. 통과 시 children 렌더링
