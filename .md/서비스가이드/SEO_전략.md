# 채널별 SEO/해시태그 전략

> 네이버 블로그 SEO 7기준 채점, 인스타그램 해시태그 전략, Threads 해시태그, YouTube SEO까지 채널별 검색 최적화 전략을 정리한다.

---

## 1. 네이버 블로그 SEO (7가지 기준)

### API: `/api/keywords/seo-score`

```
POST /api/keywords/seo-score
Content-Type: application/json
Authorization: Bearer {token}

{
  "title": "카페 인테리어 3가지 필수 팁",
  "body": "본문 HTML/마크다운 텍스트...",
  "keyword": "카페 인테리어",
  "hashtags": ["카페인테리어", "카페창업"]  // 선택
}
```

### 7가지 SEO 평가 기준

총점 100점 만점. 각 기준별 배점과 채점 기준은 아래와 같다.

#### 1) 제목 키워드 포함 (15점)

| 조건 | 점수 |
|------|------|
| 제목에 키워드 완전 포함 | 15점 |
| 키워드 부분 매칭 | (매칭 비율/전체) x 10점 |
| 키워드 미포함 | 0점 |

**실전 팁:** 제목 앞쪽에 핵심 키워드를 배치하면 네이버 검색에서 유리하다.

#### 2) 키워드 밀도 (20점)

키워드 밀도 = (키워드 등장 횟수 x 키워드 글자수) / 본문 전체 글자수 x 100

| 밀도 범위 | 점수 | 평가 |
|-----------|------|------|
| 1.5~4% | 20점 | 최적 |
| 1~1.5% 또는 4~5% | 15점 | 양호 |
| 5% 초과 | 5점 | 과다 (키워드 스터핑 위험) |
| 0% 초과~1% 미만 | 10점 | 부족 |
| 0% | 0점 | 키워드 없음 |

**최적 밀도:** 2~3%가 이상적. B2B는 5~8회, B2C는 3~5회 자연 포함을 목표로 한다.

#### 3) 콘텐츠 길이 (15점)

| 글자 수 | 점수 | 평가 |
|---------|------|------|
| 1,500~3,500자 | 15점 | 최적 (네이버 블로그 기준) |
| 1,000~1,499 또는 3,501~5,000 | 12점 | 양호 |
| 500~999 | 8점 | 짧음 |
| 500 미만 | 3점 | 매우 짧음 |

**가이드:**
- B2B 블로그: 2,500~3,500자 목표
- B2C 블로그: 1,800~2,500자 목표

#### 4) 소제목 구조 (15점)

| 조건 | 점수 |
|------|------|
| H2 2개 이상 + H3 1개 이상 | 15점 |
| H2 1개 이상 | 10점 |
| 소제목 없음 | 3점 |

**실전 팁:**
- H2는 글의 대주제 분기 (3~4개 권장)
- H3는 H2 하위 세부 포인트
- 소제목에도 키워드를 자연스럽게 포함

#### 5) 가독성 — 문단 구조 (15점)

| 조건 | 점수 |
|------|------|
| 문단 5개 이상 + 평균 문단 길이 300자 이하 | 15점 |
| 문단 3개 이상 | 10점 |
| 문단 3개 미만 | 5점 |

**실전 팁:**
- 한 문단은 3~4문장(150~300자) 이내
- 빈 줄로 문단을 명확히 구분
- 리스트/표 활용으로 가독성 향상

#### 6) 도입부 키워드 (10점)

| 조건 | 점수 |
|------|------|
| 첫 200자 내 키워드 포함 | 10점 |
| 미포함 | 0점 |

**실전 팁:** 첫 문장에 키워드를 넣으면 가장 효과적. 네이버는 도입부 키워드를 중요한 랭킹 시그널로 활용한다.

#### 7) 키워드 분포 (10점)

**복합 키워드 (2어절 이상):**
- 키워드 구성 단어들이 본문 전체에 등장하는 비율로 채점
- 예: "카페 인테리어" → "카페"와 "인테리어" 각각 확인

**단일 키워드:**
- 본문을 3등분하여 각 구간에 키워드가 분포하는지 확인
- 3구간 모두 포함: 10점, 2구간: 7점, 1구간: 3점

**실전 팁:** 키워드를 한 곳에 몰아넣지 말고 도입/본론/결론 전체에 골고루 분포시킨다.

## 2. 네이버 블로그 SEO 실전 체크리스트

### 콘텐츠 작성 시

| 항목 | 체크 |
|------|------|
| 제목에 핵심 키워드 포함 | |
| 제목 길이 25~40자 | |
| 첫 200자에 키워드 배치 | |
| H2 소제목 3~4개 사용 | |
| 본문 1,500~3,500자 | |
| 키워드 밀도 2~3% | |

## 3. 인스타그램 해시태그 전략

### 해시태그 구조 (3계층)

| 계층 | 개수 | 검색량 | 목적 |
|------|------|--------|------|
| 대형 태그 | 3~5개 | 100만+ 게시물 | 노출 볼륨 (경쟁 치열) |
| 중형 태그 | 5~10개 | 1만~100만 게시물 | 탐색탭 노출 목표 |
| 소형/니치 태그 | 5~10개 | 1만 미만 | 타겟 도달, 상위 노출 |

### B2B 인스타 해시태그 (10~15개)

```
[업계 전문] #마케팅자동화 #B2B마케팅 #SaaS #마테크 #CRM
[비즈니스 일반] #디지털전환 #비즈니스성장 #스타트업
[콘텐츠 유형] #마케팅팁 #비즈니스인사이트 #마케팅전략
```

### B2C 인스타 해시태그 (15~25개)

```
[핵심] #카페인테리어 #카페창업 #카페디자인
[확장] #인테리어팁 #소자본창업 #공간디자인 #인테리어참고
[감성] #감성카페 #카페스타그램 #인스타카페 #카페투어
[검색] #카페추천 #카페맛집 #카페오픈 #소상공인
[일반] #일상 #데일리 #추천 #꿀팁
```

### 업종별 필수 해시태그

| 업종 | B2B 태그 | B2C 태그 |
|------|---------|---------|
| 음식업 | #외식업 #F&B #프랜차이즈 | #맛집 #먹스타그램 #맛집추천 |
| 의료 | #의료마케팅 #병원마케팅 | #건강 #건강관리 #의료정보 |
| 교육 | #에듀테크 #교육사업 | #공부 #합격 #교육 |
| 뷰티 | #뷰티산업 #K뷰티 | #뷰티 #화장품 #스킨케어 |
| IT | #IT기업 #테크 #SaaS | #앱추천 #생산성 #테크 |

## 4. Threads 해시태그 전략

### B2B (3~5개)

짧고 정확한 업계 태그 위주. 너무 많으면 전문성이 떨어져 보인다.

```
#마케팅자동화 #B2B #SaaS성장
```

### B2C (5~10개)

Threads 특성상 대화형 톤에 맞는 트렌디한 태그.

```
#카페인테리어 #카페창업 #카페추천 #인테리어팁 #카페꿀팁 #소자본창업
```

## 5. 페이스북 해시태그 전략

### B2B (3~5개)

페이스북은 해시태그보다 콘텐츠 자체의 공유/반응이 중요. 최소한으로 사용.

```
#마케팅자동화 #B2B마케팅 #비즈니스
```

### B2C (3~5개)

```
#카페인테리어 #카페창업 #인테리어팁
```

## 6. YouTube SEO 전략

영상 스크립트(STEP7) 생성 시 참고하는 YouTube SEO 요소:

### 제목 최적화

| 규칙 | 예시 |
|------|------|
| 키워드를 제목 앞에 배치 | "카페 인테리어 - 매출 30% 올리는 3가지 팁" |
| 숫자/리스트 활용 | "3가지", "5분만에", "TOP 5" |
| 호기심 유발 | "이것만 바꿨는데", "절대 하지 마세요" |
| 40~60자 이내 | 검색 결과에서 잘리지 않는 길이 |

### 설명란(Description)

| 요소 | 가이드 |
|------|--------|
| 첫 2줄 | 핵심 키워드 + 영상 요약 (검색 미리보기) |
| 타임스탬프 | 씬별 챕터 구분 |
| 관련 링크 | 블로그 글, 서비스 페이지 |
| 해시태그 | 3~5개 (영상 상단 노출) |

### 썸네일 전략

| B2B | B2C |
|-----|-----|
| 데이터 숫자 강조 | 감성적 인물/장면 |
| 차분한 블루/그레이 | 밝고 대비 강한 색상 |
| 전문적 폰트 | 임팩트 있는 큰 글씨 |

## 7. 채널 간 SEO 시너지 전략

### 크로스 채널 키워드 일관성

하나의 프로젝트에서 생성된 모든 채널 콘텐츠는 동일한 `keyword`와 `fixedKeywords`를 공유한다. 이를 통해:

1. **네이버 블로그** → 메인 키워드 SEO 최적화 (본문 3~8회)
2. **인스타그램** → 해시태그로 키워드 확장 (15~25개)
3. **Threads** → 간결한 업계 태그 (3~10개)
4. **YouTube** → 제목 + 설명란 키워드 + 해시태그

### 콘텐츠 발행 순서 추천

```
1. 블로그 발행 (SEO 기반, 롱폼)
2. 인스타그램 발행 (비주얼 + 해시태그)
3. Threads 발행 (인사이트/대화형)
4. 영상 업로드 (블로그 임베드)
```

블로그에 영상을 임베드하면 네이버 블로그 SEO에 추가 가점이 된다.

## 8. 키워드 등급별 SEO 전략

| 등급 | SEO 전략 | 콘텐츠 방향 |
|------|----------|------------|
| D+~D (추천) | 키워드 집중, 빠른 상위 노출 가능 | 키워드 중심 정보형 콘텐츠 |
| C (적정) | 키워드 + 차별화 콘텐츠 | 깊이 있는 가이드, 사례 |
| B (높음) | 롱테일 키워드 확장 전략 | 시리즈 콘텐츠, 전문성 |
| A (치열) | 직접 공략 비추천 → 롱테일 우회 | 니치 키워드로 주변 공략 |
