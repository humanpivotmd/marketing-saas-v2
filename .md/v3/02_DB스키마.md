## 3. ER 관계도 (텍스트)

```
users ──┬──< projects ──┬──< image_scripts
        │               ├──< video_scripts
        │               └──< contents (project_id FK)
        ├──< contents
        ├──< keywords ──< projects (keyword_id FK)
        ├──< user_prompts
        ├──< channels
        ├──< brand_voices
        └──> industries (industry_id FK)

industries ──< industries (parent_id 자기참조, 3레벨)

projects.content_ids (JSONB) → contents.id 매핑
image_scripts.content_id → contents.id
video_scripts.content_id → contents.id
```
