## 🟡 AI 생성 파이프라인 연결 흐름

```
create/draft-info
  → POST /api/generate/titles
  → POST /api/projects
  → PATCH /api/projects/:id
  → 이동: create/generating

create/generating
  → POST /api/generate/draft
  → 이동: create/channel-write

create/channel-write
  → GET /api/projects/:id
  → POST /api/generate/pipeline (SSE)
  → POST /api/generate/single
  → PUT /api/contents/:id
  → PATCH /api/projects/:id
  → 이동: create/image-script

create/image-script
  → GET /api/projects/:id
  → POST /api/generate/image-script (채널별)
  → PATCH /api/projects/:id
  → 이동: create/video-script

create/video-script
  → GET /api/projects/:id
  → POST /api/generate/video-script
  → PATCH /api/projects/:id
  → 이동: /contents
```
