# v3 서비스 개요

## 1. 서비스 핵심 가치

### B2B/B2C 전용 콘텐츠 생성
- 사용자가 마이페이지에서 비즈니스 유형(B2B/B2C)을 선택
- B2B: 전문적/격식체, 데이터/사례/ROI 중심, 구매 담당자/임원 타겟
- B2C: 친근/감성적, 스토리텔링/공감/행동 유도, 일반 소비자 타겟
- 모든 프롬프트(제목/초안/채널변환/이미지/영상)에 B2B/B2C 분기 적용

### 채널별 맞춤 생성
- 블로그(네이버), Threads, Instagram, Facebook, 영상 스크립트
- 채널별 분량/해시태그/톤/CTA 규칙이 프롬프트에 내장
- 블로그를 먼저 생성하고, 그 결과를 소스로 SNS 채널 변환

### 커스텀 프롬프트
- 사용자가 자유 텍스트로 추가 지시를 입력
- 3가지 적용 모드: priority(최우선), combine(동등 결합), reference(참고 수준)
- 채널별(blog/instagram/threads/facebook/image/video_script) 개별 설정 가능 (마이페이지)

## 2. 7단계 플로우

| 단계 | 이름 | 설명 |
|------|------|------|
| STEP 1 | 마이페이지 설정 | B2B/B2C, 업종, 회사명, 운영 채널, 타겟, 고정키워드, 톤 설정 |
| STEP 2 | 키워드 분석 | 네이버 검색량/경쟁도/CPC 분석 후 키워드 선택 (기존 v2 기능) |
| STEP 3 | 초안 정보 입력 | 글 유형(info/intro/service/product), 톤, 커스텀 프롬프트 입력 후 제목 5개 생성 -> 선택 -> 프로젝트 생성 |
| STEP 4 | 초안 작성 | Claude API로 핵심 초안 자동 생성 (백그라운드, 유저 비노출) |
| STEP 5 | 채널별 글 생성 | 선택된 채널(blog/threads/instagram/facebook)별로 SSE 스트림으로 순차 생성. 블로그 우선 생성 후 SNS 변환 |
| STEP 6 | 이미지 프롬프트 | 채널별 이미지 프롬프트 3장 + 썸네일 1장 생성 (Midjourney/DALL-E/Stable Diffusion 등 6종 AI 도구 지원) |
| STEP 7 | 영상 스크립트 | 숏폼(세로)/일반(가로) 영상 스토리보드 생성. 장면별 visual/narration/text_overlay/transition |

## 3. v2 대비 변경점

### 신규 추가
- projects 테이블 기반 7단계 세션 관리
- B2B/B2C 분기 프롬프트 전체 적용
- industries(업종 마스터) 테이블 + 관리자 CRUD
- image_scripts 테이블 + 이미지 프롬프트 생성 API/페이지
- video_scripts 테이블 + 영상 스크립트 생성 API/페이지
- user_prompts 테이블 + 채널별 커스텀 프롬프트 (마이페이지)
- SetupRequired 컴포넌트로 마이페이지 필수 설정 강제
- FlowGuard 컴포넌트로 단계 순서 강제
- 채널별 콘텐츠 SSE 스트림 생성 (/api/generate/pipeline)
- 캘린더 페이지 (발행 예정일 기반 월간 뷰)
- 콘텐츠 목록에 프로젝트 탭 추가
- 마이페이지에 비즈니스 프로필 탭 (business) 추가

### 기존 변경
- users 테이블에 10개 컬럼 추가 (business_type, selected_channels, target_audience 등)
- contents 테이블에 project_id, confirmed_at, revision_note 추가
- contents.type에 'facebook', 'video_script' 추가
- contents.status에 'confirmed', 'edited', 'failed' 추가
- channels.platform에 'facebook' 추가
- usage_logs.action_type에 6개 추가 (draft_generate, title_generate, image_script_generate, video_script_generate, quality_score, content_revise)
- plans에 starter/premium 플랜 추가
- 랜딩 페이지 features/plans 업데이트
- 가이드 페이지에 이미지/영상 스크립트 단계 추가
